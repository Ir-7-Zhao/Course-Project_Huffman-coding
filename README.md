# 离散结构 Project-Huffman编码

本仓库用于存放大学离散结构课程实践项目的相关项目文件

## 项目目标 

完成文本、图像、音频的哈夫曼编码程序，要求根据输入中出现的频率特征，构造对应的哈夫曼编码，并比较该编码和原始编码（ASCII、BMP、WAV）的大小。

## 项目定位

1. 首先，需要明确的是本项目重在探究简单信息论和离散结构知识背景下，Huffman编码及其算法的产生缘由、基本原理、简单编程，重点不在于算法的具体实现，包含的算法实现代码比较粗浅简陋，也只会简单介绍算法代码的实现思路而非具体细节

2. 其次，本项目给出的Huffman算法实现代码由于笔者水平有限，可能存在较大漏洞，但应该可以基本实现Huffman算法的基本逻辑并简单展示Huffman编码的压缩效率，由于代码量并不大，所以代码全部写在了一个文件里，并在src文件里附上了编译好的.exe文件 ，可以直接在终端上简单运行

3. 最后，正如前面所说，本项目聚焦Huffman编码及其算法的产生缘由、基本原理、简单编程，除了实现代码外，其他主要内容都在docs文件夹里的“项目技术文档.md”里 

##  项目内容
1. docs文件夹
内含笔者所写的项目技术文档，技术文档包含算法由来，算法原理即编程实现的大致思路
2. src文件夹
内含符合项目要求的用cpp语言编写的Huffman算法实现代码及编译好的.exe程序
3. test文件夹
内含从网络上下载的简单bpm、wav测试文件，均包含原来的README.md和LICENSE，以及一个简单的用于测试的.txt文件，另外bpm和wav文件均选择一个转换为了.txt格式，以便了解相应文件的原始编码
